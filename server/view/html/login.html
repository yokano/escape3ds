ESCAPE 3DS

- 3DSブラウザ用脱出ゲーム作成ツール -
「ESCAPE 3DS」は Nintendo 3DS のウェブブラウザで遊ぶことができる脱出ゲームを無料で作成できるサービスです。パソコンやスマートフォンのウェブブラウザ上でも遊ぶことができます。作成したゲームは自由に公開、編集、配布可能です。このサービスのソースコードはすべて MIT ライセンスで GitHub 上に公開されています。このサービスは非営利の個人製作物であり、任天堂やその他の団体との関係は一切ありません。このサービスに関するお問い合わせは、画面下のお問い合わせフォームから行ってください。
